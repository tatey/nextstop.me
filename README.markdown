# Antenna Mate

## Installation

Clone the repository and bootstrap.

    $ cd nextstop.me
    $ _script/bootstrap

Serve.

    $ _script/serve

## Deploy

Configure _jekyll_s3.yml. See [jekyll-s3 for more information](https://github.com/laurilehmijoki/jekyll-s3).

Deploy

    $ _script/deploy
